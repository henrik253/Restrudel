# analysis

Corpus collection + sound-distribution analysis (roadmap Phase 1).

Fetches all available Strudel songs, parses the `.js`, and measures which
sounds/synths/effects real patterns actually use — the evidence that drives
distribution-weighted generation in `data_gen/`.

Outputs (tables + plots) are written to Drive `Restrudel/analysis/` and
summarized in `REPORT.md`. See [../roadmap.md](../roadmap.md).
